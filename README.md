# Telecom Customer Satisfaction & Churn Dashboard

## Project Overview

This Telecom Customer Satisfaction & Churn Analysis project analyzes customer satisfaction, network performance, and churn behavior across major telecom operators. The interactive Power BI dashboard provides valuable insights that help telecom companies identify factors influencing customer churn and improve customer retention strategies.

---

## Tools & Technologies

- Power BI
- Excel / CSV Dataset
- Power Query
- DAX (Calculated Columns & Measures)
- K-Means Clustering (Customer Segmentation)

---

## Dataset Description

The dataset contains customer-level information including:

- Customer ID
- Customer Name
- Telecom Operator
- Network Rating
- Speed Rating
- Customer Service Rating
- Pricing Rating
- Overall Satisfaction Rating
- Churn Status
- Customer Cluster

---

## Data Cleaning & Processing

- Removed duplicate customer records
- Handled missing and inconsistent values
- Standardized operator names and rating categories
- Created calculated measures for Customer Count, Churn Count, Churn Rate, Average Satisfaction, and Average Network Rating
- Applied K-Means clustering for customer segmentation
- Optimized data model for interactive reporting

## Key Insights

- Overall customer churn rate is **38.24%**.
- **Reliance Jio** achieved the highest customer satisfaction and network performance.
- **Bharti Airtel** recorded comparatively lower satisfaction and network ratings.
- Customer segmentation identified three distinct clusters with different satisfaction levels.
- Customers in the lowest satisfaction cluster showed the highest churn rate.
- Operator-wise analysis helps identify areas requiring service quality improvements.

---

## Conclusion

The dashboard provides a comprehensive analysis of customer satisfaction, operator performance, and churn behavior. By identifying high-risk customer segments and evaluating service quality across telecom operators, businesses can implement targeted retention strategies, enhance customer experience, and improve overall operational performance.
